# UniteAGamer

## UAG Traders Hub (ARC Raiders)

Unite A Gamer (UAG) Traders Hub is a standalone Flutter + Firebase app designed to support ARC Raiders players with:

- Blueprint tracking (Owned / Missing / Dupes / Wanted)
- Community intel and drop reporting
- Trading listings and offers
- Trade session booking and coordination
- Notification system for matches, offers, and sessions

---

## 🚧 Current Status

This is a **beta testing build**.

Core systems are implemented and functional, including:
- Authentication and user profiles
- Blueprint tracker grid
- Trading flow (Listings → Offers → Sessions)
- In-app notifications
- Availability and region-based matching

Some features (such as automated push notifications and iOS support) are still being finalised.

## 🔥 Key Features

### Blueprint Tracker
Track your ARC Raiders blueprints with:
- Owned
- Missing
- Duplicate tracking
- Wanted system

### Trading System
- Create listings
- Send and receive offers
- Accept / decline trades
- Auto-create trade sessions

### Trade Sessions
- Schedule trades
- Set availability
- Coordinate with other players
- Share session details

### Smart Matching (Beta)
- Matches based on:
  - Blueprint needs and duplicates
  - Region compatibility
  - Availability overlap
